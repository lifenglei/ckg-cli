

#### 脚手架使用说明
> 描述：通过脚手架可以快速构建项目 节约开发成本 方便二次开发 
#### 全局安装ckg-cli
```
    npm install ckg-cli -g
```
#### 查看可用模板
```
    ckg-cli list
```
